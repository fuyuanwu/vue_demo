这里所以的方法挂载在全局对象window[RO](FrontBase)上面：

这些方法只做了迁移，功能未全部调整。接下来会调整，有需要的使用务必调整功能正常，或者寻求帮助。

引入方式：
    页面js模块中引人base.js(可以是本文件夹内的base.js,也可以是单个项目中自定义的base.js-需声明全局对象FB)
    然后引人对应的模块
    ** 可以将多个模块和base打包成一个，也可以将模块单独打包进页面js **
调用方式
    例:window[RO].getUrlParam('lang')

扩展方法：
    $.extend(window[RO],{
        XX:function(){
            //some code
        }
    })
对于后期加入的扩展的方法务必加入注释
/*
    功能：********
    参数：********(若有)
    调用方法：****
    作者：********
*/
